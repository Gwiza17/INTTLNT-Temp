# INTTLNT MVP

Global Mobility Intelligence for Critical Skills.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials
4. Run the development server: `npm run dev`

## Database Setup

This project uses Supabase. Run the migrations in `supabase/migrations` on your Supabase project to set up the database schema.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (Auth, Postgres, Storage)
- Vercel (hosting)
